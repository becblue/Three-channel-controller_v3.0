# 三通道控制器重构工作区

日期：2026-06-16

这个目录是三通道控制器固件的独立重构工作区。

原始 STM32/Keil 工程保留在本目录之外，作为行为参考和回退版本。重构过程中，新文档、新代码、新模块一律放在本目录内，避免和旧项目混在一起。

## 工作区规则

- 架构文档放在 `docs/`。
- 新固件头文件放在 `Core/Inc/`。
- 新固件源文件放在 `Core/Src/`。
- 不向原始工程目录添加重构文件。
- 不在原始工程中混入新架构代码。
- 所有 `.md` 文档使用中文。
- 所有代码注释使用中文。
- 所有新增文件统一使用 UTF-8 编码。

## 重构目标

把当前多轮修改后耦合严重的应用层代码，重构为边界清晰、单向依赖、方便验证的固件结构。

目标模块包括：

- 板级 IO 层
- 输入管理
- 通道控制
- 继电器管理
- 反馈监控
- 安全报警管理
- 外部电源监控
- 温度管理
- 报警输出
- 显示管理
- 看门狗管理
- 维护服务
